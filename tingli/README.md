# 睡前听力 - 磨耳朵英语学习 App

一款专为英语学习者设计的睡前听力应用，帮助你在轻松的氛围中提升英语听力水平。

## 🌟 核心功能

- **音乐播放器风格** - 音频采用优雅的圆形胶片设计
- **倍速播放** - 支持 0.5x-2.0x 多档调速
- **顺序播放** - 自动按 U1→U6→ACT 顺序播放
- **字幕跳转** - 点击字幕快速定位
- **睡前定时** - 10/20/30分钟自动停止
- **每日打卡** - 记录播放时长，达标提醒
- **位置记忆** - 继续上次播放位置
- **自动更新** - 检测新版本并一键安装

```bash

# 或使用自动化脚本
./cicd/build_and_upload_key.sh
```
